# The Bridge of Death

A suspense-filled desktop game built with Java Swing, inspired by the iconic Glass Bridge challenge. Every step could be your last — choose the right glass to survive, or watch it shatter beneath you.

## Gameplay

- You stand before a bridge made of glass panels
- At each step, two glass panels appear — only one is safe
- Choose the right one and advance to the next step
- Choose the wrong one and the glass shatters, restarting the game
- Make it across all steps to win

## Features

- Random glass selection on every run — no two games are the same
- Suspenseful decision-making gameplay
- Clean desktop UI built with Java Swing
- Instant restart on wrong choice

## Tech Stack

- Language: Java
- UI Framework: Java Swing
